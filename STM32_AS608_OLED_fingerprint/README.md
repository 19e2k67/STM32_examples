# STM32_AS608_OLED
STM32F103C8T6 AS608 OLED 寄存器版本 Keil uVision5  
AS608  
3.3V-->3.3V  
TX-->PB11  
RX-->PB10  
GND-->GND  

OLED  
GND-->GND  
VCC-->3.3或5V  
SCL-->PB8  
SDA-->PB9  

KEY1-->PA1  
KEY2-->PA2  
KEY3-->PA3  
KEY4-->PA4  
KEY5-->PA5
